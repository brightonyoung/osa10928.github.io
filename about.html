Hello, I'm Stephen Agwu, a self-taught UX/UI FrontEnd Webdeveloper. I relish the chance to learn new and poingant lessons, not just in software development, but life in general. Feel free to navigate this sites links to learn just a little bit more about me and my work.

About Website

This website was my first extensive personal project. It was built using jekyll on the back and and hosted by GitHub as my personal page. As such all the code can be seen at this pages repository.
I considered building my own site but didn't for one main reason: I wanted this site to be a showcase of my Frontend skills and purely my Frontend skills. This means I wanted to not only remove the need for time consuming Backend implimentations, but that I wanted also to be confined to a certain backend implimentation. Not often does a Frontend developer get to choose how his work is served and by challenging myself to fit within the jekyll environment I hoped to increase my Frontend flexibility.
All in all, it was actually pretty fun making and designing this website. Not only was it remenicient of my time spent with Ruby on Rails using Frontend tools like layouts and includes, it also had site and page variables reminicient of RoR variables in helper and control classes. This made work with jekyll highly productive and conducive to proving high levels of organisation. I would recommend jekyll to other Frontend Developers looking to get started with a page of their own. (But honestly, I can't wait to launch a site myself from beginning to end.)
