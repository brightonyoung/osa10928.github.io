About Website

This website was my first extensive personal project. It was built using jekyll and hosted by github as my personal github page (gh-page). As such all the code can be seen at this page's repository.
I considered building a full stacks site using Ruby on Rails and hosted with heroku, but didn't for one main reason: I wanted this site to be a pure showcase of my Frontend skills. In order to do that I needed to be confined to a certain backend implimentations (see jekyll instructions). Not often does a Frontend developer get to choose exactly how their work is served and by challenging myself to fit within the jekyll environment I hoped to increase my Frontend flexibility.
All in all, it was actually pretty fun making and designing this website. Not only was it remenicient of my time spent with Ruby on Rails using Frontend tools like layouts and includes, it also had site and page variables reminicient of RoR variables in helper and control classes. This made work with jekyll highly productive and conducive to proving high levels of organization. I would recommend jekyll to other Frontend Developers looking to get started with a page of their own. (But honestly, I can't wait to launch a fullstacks site, as becoming a fullstacks developer is my next goal.)
